hexo个人博客搭建

按照官网文档提示，安装以下两个依赖项：

nodejs、git

安装完毕后可以通过

进行验证

创建并安装基础的博客

创建一个用于搭建博客的文件夹，

在文件夹通过

右键 >>> “open git bash here”

打开git终端

而后，进行hexo的基础安装

并通过

验证

克隆仓库

从github上克隆hexo的官方模板仓库

安装npm

本地运行

键入hexo server，成功启动

而后，在浏览器输入url为127.0.0.1:4000的本地4000端口，成功在本地运行个人hexo博客

基础搭建完成

发布新文章

然后我们就会收获到一个名为“好好学习天天向上“的文章

可以打开看一下，注意⚠️title，date等的冒号后面要敲一个空格（特殊格式）

hexo s 后可在浏览器看到刚刚发布的新文章
